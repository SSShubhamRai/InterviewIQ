# 🎤 AI Interview Agent

An AI-powered mock interview platform that helps users prepare for job interviews through personalized AI-generated questions, voice/text interaction, and real-time performance evaluation.

## 🚀 Live Demo

🔗 https://interviewiq-l227.onrender.com/

## 📂 GitHub Repository

🔗 https://github.com/SSShubhamRai/InterviewIQ

## ✨ Features

- Resume-based AI interview question generation
- Voice and text-based interview interaction
- Real-time interview feedback and evaluation
- Performance score and improvement suggestions
- Interview history tracking
- Firebase Google Authentication
- Razorpay payment integration
- Responsive UI using Framer Motion

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- Firebase Authentication
- Google Sign-In

### Payments
- Razorpay

### AI Integration
- Gemini AI

### Deployment
- Render
